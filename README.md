### 設定

プロジェクトには.env を作成し
firebase の管理画面から歯車マーク ⇨ 表示されている情報を以下のように貼り付けましょう！

```
//.envに記述する内容 xxxの箇所はfirebase記載のものを記述
VITE_apiKey="xxxxx"
VITE_authDomain="xxxxx"
VITE_projectId="xxxxx"
VITE_storageBucket="xxxxx"
VITE_messagingSenderId="xxxxx"
VITE_appId="xxxxx"
```

### npm のインストール

`npm i firebase`

`npm i react-router-dom`

をインストールしておく
